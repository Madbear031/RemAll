# **1.Добавление новой задачи**
## **Порядок действий**
  1. Открыть приложение RemAll на мобильном устройстве
  2. Нажать на кнопку в виде знака « + » (в правом верхнем углу экрана)
  3. Ввести название задачи
  4. Выбрать один из 3 вариантов приоритета(зеленый – низкий, желтый – средний, красный – высокий)
  5. Нажать на кнопку «добавить дату»
  6. Выбрать нужную дату
  7. Нажать на кнопку «добавить время»
  8. Выбрать нужное время
  9. Нажать на кнопку «добавить задачу»
  ## **Ожидаемый результат**
   Добавление новой задачи к списку уже существующих задач
  ## **Фактический результат**
   Перенаправление на главную страницу с задачами и отображение новой задачи.
  ## **Оценка (Пройден/провален)**
   Пройден
